CSSVoçê já viu o seletor BACKGROUND-SIZE?

Conheça um pouco sobre a TAG

A propriedade background-size permite que você altere o tamanho da imagem de fundo para melhor atender sua página. Será possível especificar em termos de porcentagem ou pixels quão grande uma imagem de fundo deve ser. Isto irá permitir a reutilização de imagens em vários contextos diferentes e também expandir um fundo para preencher uma área com mais precisão.
