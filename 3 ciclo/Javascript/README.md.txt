# Jquery, é uma biblioteca do javascript, para facilitar o manuseio do DOM

## base para o JQuery:
	
		**$(seletor).evento ( acao );**
			|		 |       |	
		
	SELETOR: String que representa qual elemento DOM(Document Object Model) a ser manipulado

	EVENTO: Dispara a acao sob algum evento.

	ACAO: funcao a ser executada

	Seleciona alguns Objetos DOM (TAGs) e a partir do evento será dispara a ação.
